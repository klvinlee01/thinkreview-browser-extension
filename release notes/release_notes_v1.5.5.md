# ThinkReview - Version 1.5.5 Release Notes

**Release Date:** 1 March 2026

---

## 🎯 What's New

### Code Suggestions – In the tab and in GitLab diffs 💡

**AI code suggestions** are now available in two places:

- **Code Suggestions tab:** View AI-generated code improvements in the integrated review panel. Each suggestion includes copy-to-clipboard support and an **Explain** button to ask the AI for more context in the conversation.
- **GitLab diff injection (Beta):** Optionally show suggestions directly inline in your GitLab merge request diffs. Toggle **"Show suggestions in GitLab diff"** in the Code Suggestions tab—off by default, so you can enable it when you want suggestions right in the file view.

Both features are available for **Professional**, **Lite**, and **Teams** subscribers.

---

## 🔧 Improvements

- **GitLab injection default:** The "Show suggestions in GitLab diff" toggle now defaults to **off**, so you opt in when you want inline suggestions.
- **Code Suggestions UI:** Compacted the GitLab injection toggle for a cleaner look in the Code Suggestions tab.
